# Netflix Clone

A modern Netflix clone built with React, Vite, and Tailwind CSS.

## Features

- 🎬 Movie/TV Show browsing
- 🎨 Netflix-like UI/UX
- 📱 Responsive design
- ⚡ Fast performance with Vite
- 🎭 Hover effects and animations

## Tech Stack

- React 18
- React Router
- Tailwind CSS
- Vite

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

## Build for Production

```bash
npm run build
```
